# athrillとは
Athrill は CPU エミュレータです. 
Athrill を使用すると，仮想マイコン上で組込みプログラムを簡単に実行/デバッグすることができます．
ベアメタルプログラムやリアルタイムOS上で動作する組込み制御プログラムを評価できます．
